# scope of the project: 
the project focuses on building a python application that generates passwords,with costomizable length and key features such as :- 
- Generates secure, customizable passwords using a GUI built with Tkinter.
- Offers a unique Memorability Mode that allows users to include personal details to remember the generated password without compromising password strength.
- Evaluates password strength with feedback indicators (Weak / Moderate / Strong).
- multilingual passwords can be generated.user can select the language mode such as hindi, english or both.
- Provides a beginner-friendly interface suitable for general users.
- Serves as an educational tool to promote password hygiene and secure coding practices.
